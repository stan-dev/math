Detailed Description

template<typename F, typename Tyy, typename Typ, typename... T_par>
class stan::math::dae_system< F, Tyy, Typ, T_par >

IDAS DAE system that contains information on residual equation functor, sensitivity residual equation functor, as well as initial conditions.

This is a base type that is intended to contain common values used by forward sensitivity system.

Template Parameters
Ftype of functor for DAE residual
Tyyscalar type of initial unknown values
Typscalar type of initial unknown's derivative values
Tparscalar type of parameters
