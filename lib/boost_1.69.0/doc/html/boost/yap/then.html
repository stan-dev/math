Function template then

boost::yap::then

Synopsis

// In header: <boost/yap/algorithm.hpp>


template<typename Expr> decltype(auto) then(Expr && expr);

Description

Returns the then-expression in an if_else expression.

Equivalent to get(expr, 1_c).

Note

then() is only valid if Expr is an expr_kind::if_else expression.
