Spawn a resumable coroutine. 

template<
    typename T,
    typename Executor,
    typename CompletionToken>
DEDUCED co_spawn(
    coro< void, T, Executor > c,

This function spawns the coroutine for execution on its executor. It binds the lifetime of the coroutine to the executor.
