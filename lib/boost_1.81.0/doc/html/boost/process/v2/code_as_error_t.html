Description

Convert the exit-code in a completion into an error if the actual error isn't set. 

process proc{ctx, "exit", {"1"}};

proc.async_wait(code_as_error(
   [](error_code ec)
   {
     assert(ec.value() == 10);
     assert(ec.category() == error::get_exit_code_category());
   }));
