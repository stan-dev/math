authority_view::size

Return the number of characters in the authority.

Synopsis 
std::size_t
size() const;
Description

This function returns the number of characters in the authority.

Example 
assert( authority_view( "user:pass@www.example.com:8080" ).size() == 30 );
Exception Safety

Throws nothing.
