params_encoded_ref::empty

(Inherited from params_encoded_base)

Return true if there are no params.

Synopsis 
bool
empty() const;
Example 
assert( ! url_view( "?key=value" ).encoded_params().empty() );
Complexity

Constant.

Exception Safety

Throws nothing.
