pct_string_view::decoded_size

Return the decoded size.

Synopsis 
std::size_t
decoded_size() const;
Description

This function returns the number of characters in the resulting string if percent escapes were converted into ordinary characters.

Complexity

Constant.

Exception Safety

Throws nothing.
