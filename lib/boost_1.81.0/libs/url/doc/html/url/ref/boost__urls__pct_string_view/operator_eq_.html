pct_string_view::operator=

Assignment.

Synopsis 
pct_string_view&
operator=(
    pct_string_view const& other);
Description

The copy references the same underlying character buffer. Ownership is not transferred.

Postconditions 
this->data() == other.data()
Complexity

Constant.

Exception Safety

Throws nothing.

other The string to copy.
