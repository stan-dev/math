reserved_chars

The reserved character set.

Synopsis

Defined in header <boost/url/rfc/reserved_chars.hpp> 

constexpr auto reserved_chars = ~unreserved_chars;
Example

Character sets are used with rules and the functions grammar::find_if and grammar::find_if_not. 

result< decode_view > rv = grammar::parse( "Program%20Files", pct_encoded_rule( reserved_chars ) );
Specification
  • 2.3. Unreserved Characters (rfc3986)
See Also

grammar::find_if, grammar::find_if_not, grammar::parse, pct_encoded_rule.
