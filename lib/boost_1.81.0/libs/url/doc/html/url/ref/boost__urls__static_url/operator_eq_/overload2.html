static_url::operator= (2 of 2 overloads)

Assignment.

Synopsis 
static_url&
operator=(
    url_view_base const& u);
Description

The contents of u are copied and the previous contents of this are discarded.

Postconditions 
this->buffer() == u.buffer() && this->buffer().data() != u.buffer().data()
Complexity

Linear in u.size().

Exception Safety

Strong guarantee. Exception thrown if capacity exceeded.

Exceptions

Type

Thrown On

system_error

Capacity would be exceeded.

Parameters

Name

Description

u

The url to copy.
