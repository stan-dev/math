static_url_base::normalize_scheme

(Inherited from url_base)

Normalize the URL scheme.

Synopsis 
url_base&
normalize_scheme();
Description

Applies Syntax-based normalization to the URL scheme. The scheme is normalized to lowercase.

Exception Safety

Strong guarantee. Calls to allocate may throw.

Specification
  • 6.2.2 Syntax-Based Normalization (rfc3986)
