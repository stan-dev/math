sub_delim_chars

The sub-delims character set.

Synopsis

Defined in header <boost/url/rfc/sub_delim_chars.hpp> 

constexpr grammar::lut_chars sub_delim_chars = "!$&()*+,;=\x27";
Example

Character sets are used with rules and the functions grammar::find_if and grammar::find_if_not. 

result< decode_view > = grammar::parse( "Program%20Files", pct_encoded_rule( sub_delim_chars ) );
BNF 
sub-delims    = "!" / "$" / "&" / "'" / "(" / ")"
              / "*" / "+" / "," / ";" / "="
Specification
  • 2.2. Reserved Characters (rfc3986)
See Also

grammar::find_if, grammar::find_if_not, grammar::parse, pct_encoded_rule.
