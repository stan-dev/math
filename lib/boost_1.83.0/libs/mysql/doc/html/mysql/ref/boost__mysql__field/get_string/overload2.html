field::get_string (2 of 2 overloads)

Retrieves a reference to the underlying std::string value (unchecked access).

Synopsis 
std::string&
get_string();
Preconditions

this->is_string() == true (if violated, results in undefined behavior).

Exception safety

No-throw guarantee.

Object lifetimes

The returned reference is valid as long as *this is alive and no function that invalidates references is called on *this.
