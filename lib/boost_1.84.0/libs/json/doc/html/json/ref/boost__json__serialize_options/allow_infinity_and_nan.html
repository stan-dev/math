serialize_options::allow_infinity_and_nan

Non-standard extension option.

Synopsis 
bool allow_infinity_and_nan = false;
Description

Output Infinity, -Infinity and NaN for positive infinity, negative infinity, and "not a number" doubles.

See Also

serialize, serializer.
