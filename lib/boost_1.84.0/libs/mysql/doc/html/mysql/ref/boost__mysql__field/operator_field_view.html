field::operator field_view

Constructs a field_view pointing to *this.

Synopsis 
operator field_view() const;
Description

The resulting field_view has the same kind and value as *this.

Exception safety

No-throw guarantee.

Object lifetimes

The returned object acts as a reference to *this, and will be valid as long as *this is alive.
