metadata::column_collation

Returns the ID of the collation that fields belonging to this column use.

Synopsis 
std::uint16_t
column_collation() const;
Description

This is not the collation used when defining the column in a CREATE TABLE statement, but the collation that fields that belong to this column and are sent to the client have. It usually matches the connection's collation.

Exception safety

No-throw guarantee.
