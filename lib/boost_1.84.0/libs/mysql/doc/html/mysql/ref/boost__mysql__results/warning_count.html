results::warning_count

Returns the number of warnings produced by the executed SQL statement.

Synopsis 
unsigned
warning_count() const;
Description

For operations returning more than one resultset, returns the first resultset's warning count.

Preconditions

this->has_value() == true

Exception safety

No-throw guarantee.

Complexity

Constant.
