static_execution_state::should_read_rows

Returns whether the next operation should be read some rows.

Synopsis 
bool
should_read_rows() const;
Description

Call connection::read_some_rows or its async counterpart to move forward. Metadata for the current resultset is available in this state.

Exception safety

No-throw guarantee.
