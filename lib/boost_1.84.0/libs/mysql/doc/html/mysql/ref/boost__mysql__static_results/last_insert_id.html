static_results::last_insert_id

Returns the last insert ID produced by the executed SQL statement.

Synopsis 
template<
    std::size_t I = 0>
std::uint64_t
last_insert_id() const;
Template Parameters

Type

Description

I

Resultset index. For operations returning more than one resultset, you can explicitly specify this parameter to obtain the last insert ID for the i-th resultset. If left unspecified, the last insert ID for the first resultset is returned.

Preconditions

this->has_value() == true

Exception safety

No-throw guarantee.

Complexity

Constant.
